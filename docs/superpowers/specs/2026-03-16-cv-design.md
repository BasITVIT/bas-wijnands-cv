## Layout

Twee-koloms, A4-formaat (210mm × 297mm):

| Kolom | Breedte | Inhoud |
|-------|---------|--------|
| Links | ~30% | Foto, personalia, opleidingen, cursussen, talen, vaardigheden, tech-skills |
| Rechts | ~70% | Naam + titel, profieltekst, werkervaring (successen geïntegreerd) |

**Diagonale snede:** implementeren via `clip-path: polygon(0 0, 100% 0, 85% 100%, 0 100%)` op de linkerkolom. In `@media print` terugvallen op een rechte rand (clip-path verwijderen, border-right gebruiken) voor print-compatibiliteit.

---

## Print-specificaties

- Exacte A4-afmetingen in print: `@page { size: A4; margin: 0; }`
- Achtergrondkleuren geforceerd: `-webkit-print-color-adjust: exact; print-color-adjust: exact`
- Diagonale snede vervangen door rechte rand in print (clip-path: none; border-right: 3px solid #00C2CB)
- Page-breaks voorkomen midden in werkervaringsblokken: `page-break-inside: avoid`
- Google Fonts via `<link>` — fallback font: Arial, sans-serif
- **Meerdere pagina's:** als content over 2 pagina's loopt, blijft de linkerkolom doorlopen met dezelfde navy achtergrond. De linkerkolom krijgt `min-height: 100vh` zodat hij de volle hoogte vult. Bij pagina-overloop wordt de border-right op de linkerkolom op beide pagina's zichtbaar via `box-decoration-break: clone`

---

## Foto

- `<img src="foto.jpg" alt="Bas Wijnands">` — placeholder; gebruiker plaatst zelf `foto.jpg` naast het HTML-bestand
- Cirkel-uitsnede via `border-radius: 50%`, breedte 120px
- **Fallback als foto.jpg ontbreekt:** gekleurde cirkel (#00C2CB) met initialen "BW" in wit, geïmplementeerd via CSS `::before` op een div met `onerror` handler op de img

## CSS Layout

- **Mechanisme:** CSS Grid (`display: grid; grid-template-columns: 30% 70%`)
- Compatibel met Chrome, Edge en Firefox voor print
- Fallback: Flexbox als Grid niet beschikbaar is (moderne browsers ondersteunen beide)
