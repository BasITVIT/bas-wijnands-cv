# CV HTML Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Één HTML-bestand bouwen dat een creatief, opvallend CV weergeeft voor Bas Wijnands en direct als A4-PDF printbaar is via de browser.

**Architecture:** Enkelvoudig HTML-bestand met embedded CSS. CSS Grid voor twee-kolomsindeling (30/70). Linkerkolom navy met diagonale clip-path. `@media print` stylesheet dwingt A4-formaat en achtergrondkleuren af. Geen JavaScript, geen externe assets behalve Google Fonts.

**Tech Stack:** HTML5, CSS3 (Grid, clip-path, @media print), Google Fonts (Inter), browser print (Chrome/Edge/Firefox)

**Spec:** `docs/superpowers/specs/2026-03-16-cv-design.md`
